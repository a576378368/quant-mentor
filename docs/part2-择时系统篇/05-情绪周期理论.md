# 第5章 情绪周期理论：冰点、高潮与市场轮回

## 5.1 什么是情绪周期？

情绪周期是市场情绪从低迷到高涨，再到回落的完整循环过程。它是股票市场运行的基本规律之一，反映了投资者情绪的起伏变化。理解情绪周期，就像理解一年四季的轮回一样重要——有冬天的沉寂，才有春天的萌发；有夏天的狂热，才有秋天的收获。

**炒股养家**作为A股市场情绪周期理论的奠基人，提出了经典的8阶段情绪周期模型：

```
酝酿期 → 起步期 → 加速期 → 高潮期 → 发散期 → 退潮期 → 淘汰期 → 新周期（回到起点）
```

这个8阶段模型描述了市场情绪从冰点到沸点再到冰点的完整循环。每一次循环都是一次财富重新分配的过程，理解的投资者在周期中获利，不理解的投资者在周期中亏损。

### 5.1.1 情绪周期的重要性

**炒股养家**的核心观点：

> 强势阶段：买最强龙头，敢千高位
> 弱势阶段：低吸超跌，快进快出
> 淘汰阶段：空仓等待新周期

这段话精辟地说明了情绪周期与操作策略的关系。不同的情绪周期阶段，需要采用不同的操作策略：

**在情绪周期的高潮期**，市场赚钱效应达到顶峰，龙头股持续涨停，此时应该满仓买最强龙头，享受主升浪带来的超额收益。

**在情绪周期的冰点和淘汰期**，市场亏钱效应明显，亏钱效应扩散，此时应该空仓等待新周期，不逆势操作。

**在情绪周期的起步期和加速期**，是新题材和新龙头的孕育期，此时应该小仓试错，确认后加仓。

**在情绪周期的退潮期**，市场情绪开始转弱，赚钱效应递减，此时应该快进快出，不恋战。

情绪周期决定了市场整体的赚钱效应和亏钱效应，进而决定了仓位管理和选股策略。因此，理解并正确判断当前所处的情绪周期阶段，是制定正确交易策略的前提。

**涅盘重升**是新生代游资代表人物，他对情绪演变的认知有着独到的见解。涅盘重升认为，情绪周期的本质是市场参与者的心理波动，而这种波动呈现出周期性特征。他表示："情绪周期就像潮汐，有涨潮、有退潮、有平潮。理解了这个规律，就能在正确的时机做正确的事情。"

涅盘重升特别强调，情绪周期的每个阶段都有其独特的特征，只要能够准确识别当前阶段，就能制定相应的操作策略。他说："高手与散户的区别，就在于高手能够准确识别情绪周期所处的阶段，而散户往往在错误的时间做错误的事情。"

### 5.1.2 情绪周期的时间特征

情绪周期的时间长度并非固定不变，而是受多种因素影响，包括市场环境、题材大小、资金面状况等。根据历史数据统计，一个完整的情绪周期通常持续1-3个月。

**短期题材的周期特征：**

- 周期时长：通常为2-4周
- 上涨阶段：1-2周
- 下跌阶段：1-2周
- 典型案例：消息刺激型题材、短期政策题材

**中期题材的周期特征：**

- 周期时长：通常为1-2个月
- 上涨阶段：2-4周
- 下跌阶段：2-4周
- 典型案例：行业景气周期、业绩预期型题材

**长期题材的周期特征：**

- 周期时长：通常为3个月以上
- 上涨阶段：1-2个月
- 下跌阶段：1-2个月
- 典型案例：产业革命型题材、革命性技术创新

**涅盘重升的时间周期理论：**

涅盘重升认为，情绪周期的时间长度与题材的大小成正比。他表示："大题材对应长周期，小题材对应短周期。但无论周期长短，上涨阶段和下跌阶段的时间比例大约是1:1。"

涅盘重升还发现，情绪周期可能出现"加速"或"减速"的情况。他说："当市场情绪极度亢奋时，周期可能加速，原本一个月的行情可能在两周内走完；当市场情绪低迷时，周期可能减速，原本两周的行情可能拖到一个半月。"

需要特别注意的是，情绪周期的阶段转换并非突变，而是渐进式的。在实际市场中，可能出现以下情况：

第一，周期加速。当市场情绪高涨、资金充裕、题材足够大时，情绪周期的各个阶段可能加速演进，例如原本持续3天的起步期可能缩短为1天。

第二，周期延长。当市场情绪低迷、场外资金不足、题材缺乏新意时，情绪周期的各个阶段可能延长，例如原本持续1周的加速期可能延长为2周。

第三，二次启动。某些大题材可能在第一次高潮后，经过短暂调整，再次启动第二波行情，形成"双峰"结构。

### 5.1.3 情绪周期的资金特征

情绪周期的演进本质上是资金运动的过程。在不同的情绪周期阶段，资金表现出不同的特征：

**涅盘重升的资金流动理论：**

涅盘重升对情绪周期中资金流动的规律有着深刻的研究。他将资金分为三类：先知先觉资金、后知后觉资金、不知不觉资金。

"先知先觉资金是最聪明的资金，他们在行情启动前就已经进场埋伏。后知后觉资金在行情启动后跟进，是行情的主要推动力。不知不觉资金在行情的高潮期进场，是最后的接盘侠。"

在不同的情绪周期阶段，这三类资金的进出特征如下：

**在情绪周期的酝酿期**，先知先觉资金开始试探性建仓，但大多数资金处于观望状态。涅盘重升指出："酝酿期是先知先觉资金布局的时期，他们会在这个时候悄然进场，但不会引起市场注意。"

**在情绪周期的起步期**，先知先觉资金加仓，后知后觉资金开始试探性入场。成交量温和放大，涨停股票开始增加。

**在情绪周期的加速期**，赚钱效应扩散，后知后觉资金加速进场。场外资金跑步入场，成交量持续放大，涨停股票数量大幅增加。

**在情绪周期的高潮期**，不知不觉资金在赚钱效应的吸引下大量进场，成为最终的接盘者。成交量达到峰值，涨停股票数量创历史新高。涅盘重升警示："高潮期是散户进场的高峰，也是先知先觉资金撤退的时期。"

**在情绪周期的发散期**，先知先觉资金开始撤离，后知后觉资金还在进场。资金出现分歧，成交量维持在高位但不再放大。

**在情绪周期的退潮期**，资金开始撤离市场，成交量逐步萎缩，涨停股票数量减少，跌停股票数量增加。

**在情绪周期的淘汰期**，恐慌资金不计成本出逃，成交量可能短暂放大后萎缩，跌停股票数量大幅增加。

**在情绪周期的新周期起点**，先知先觉资金再次入场试探，成交量开始温和放大。

理解资金在情绪周期各阶段的流动特征，有助于投资者判断当前处于哪个阶段，从而制定正确的操作策略。

## 5.2 情绪周期8阶段详解

### 5.2.1 酝酿阶段（冰点）

酝酿阶段是情绪周期的最寒冷时期，也是新周期的孕育期。这个阶段的市场特征是：情绪极度低迷，赚钱效应几乎消失，亏钱效应显著，成交量萎缩，市场冷清。

**涅盘重升的冰点识别法：**

涅盘重升认为，冰点是情绪周期中最重要的节点，也是逆向布局的最佳时机。他总结出冰点的几个典型特征：

"第一，涨停数量极少，市场几乎没有亮点。第二，跌停数量众多，高位股持续补跌。第三，成交量萎缩，市场参与度极低。第四，投资者情绪极度悲观，悲观言论充斥网络。第五，媒体不再报道股市相关新闻。"

**酝酿阶段的市场特征：**

第一，涨停数量极少。每日涨停股票数量可能不足10只，甚至出现个位数的情况。市场缺乏热点，资金没有进攻方向。

第二，跌停数量众多。每日跌停股票数量可能超过50只，高位股、跟风股纷纷补跌，亏钱效应显著。

第三，连板数量极少。连续涨停的股票非常稀少，即使有股票涨停，也大多是首板就结束，连板概率极低。

第四，涨跌比极低。上涨股票数量远少于下跌股票数量，涨跌比可能低于0.5，市场呈现普跌格局。

第五，成交量萎缩。市场参与度低，成交金额较高峰期萎缩30%-50%，资金观望情绪浓厚。

第六，炸板率高。涨停股票打开涨停的概率很高，即使封住涨停也容易被砸开，炸板率可能超过60%。

**酝酿阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | < 10 | 每日涨停不足10只 |
| 跌停数 | > 50 | 每日跌停超过50只 |
| 连板数 | < 3 | 最高连板不超过3板 |
| 涨跌比 | < 0.5 | 上涨股票数量远少于下跌 |
| 炸板率 | > 60% | 过半涨停股炸板 |
| 成交额 | 较高峰萎缩50% | 市场成交极度萎缩 |

**涅盘重升的操作策略：**

涅盘重升在酝酿阶段的核心策略是"关注而不操作"。他表示："冰点时期，最重要的是保持对市场的关注，寻找新周期的启动信号。但操作上要极为谨慎，因为冰点可能持续很长时间。"

具体要点如下：

第一，观察新题材的苗头。涅盘重升说："冰点时期，虽然不能操作，但一定要保持关注。很多大牛股就是在冰点时期启动的。"

第二，等待标志性信号。涅盘重升认为，新周期启动的标志性信号包括：连续涨停的股票出现、成交量开始放大、板块开始联动等。

第三，不抄底。涅盘重升特别强调不要抄底，他说："冰点下面还有冰点，盲目抄底可能让你亏损惨重。"

第四，保持耐心。涅盘重升说："冰点时期，最重要的是耐心。等待是一件很难的事情，但这是专业投资者的基本功。"

### 5.2.2 起步阶段（回暖）

起步阶段是新周期的启动期，是资金开始试探性入场的时期。这个阶段的市场特征是：新题材出现，龙头股启动，市场情绪开始转暖。

**涅盘重升的起步期识别：**

涅盘重升认为，起步期是情绪周期的关键转折点。他总结出起步期的几个典型特征：

"第一，出现新的题材或概念，这些题材具有新颖性和想象空间。第二，有股票率先涨停，成为新龙头的候选者。第三，涨停数量开始增加，跌停数量开始减少。第四，成交量温和放大。第五，板块开始出现联动效应。"

**起步阶段的市场特征：**

第一，新题材出现。市场中出现新的题材或概念，这些题材可能是政策利好、行业新闻、公司公告等。新的题材带来新的想象空间，吸引资金关注。

第二，龙头股启动。某个股票率先涨停，成为新题材的龙头股。这个龙头股通常是第一个涨停的股票，具有先发优势。

第三，少数资金介入。只有少数先知先觉的资金开始试探性买入，大多数资金仍在观望。

第四，成交量温和放大。成交量较冰点时期温和放大，但放大程度有限，主要是场内资金调仓所致。

第五，涨停数量开始增加。从冰点时的不足10只增加到10-30只，市场活跃度有所提升。

第六，板块开始联动。龙头股涨停后，同题材的股票开始跟涨，板块联动效应开始出现。

**起步阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | 10-30 | 每日涨停10-30只 |
| 跌停数 | 20-50 | 每日跌停20-50只 |
| 连板数 | 3-5 | 最高连板为3-5板 |
| 涨跌比 | 0.5-1 | 上涨与下跌股票数量基本持平 |
| 炸板率 | 40-60% | 炸板率仍然较高但有所下降 |
| 成交额 | 较冰点放大20-30% | 市场活跃度开始提升 |

**起步阶段的核心任务：识别新题材和新龙头**

起步阶段最重要的任务是识别即将成为市场主线的新题材，以及可能成为龙头股的股票。以下是识别方法：

第一，观察首板股票。涅盘重升说："我每天都会关注首板涨停的股票，找出最早涨停的那几只。这些股票可能是新龙头的候选者。"

第二，观察题材新颖度。新题材必须具备新颖性，如果是老题材的重复炒作，往往持续性较差。涅盘重升认为，新题材的"新"体现在三个方面：新技术、新政策、新需求。只有新东西才能引发市场的想象空间。

第三，观察板块联动性。龙头股涨停后，观察同板块股票的跟涨情况，如果跟风积极，说明板块联动性强。

第四，观察成交量。龙头股的成交量必须明显放大，表明有资金积极参与。

第五，观察龙虎榜。如果有知名游资席位出现在龙虎榜，说明市场认可该股票的价值。

**涅盘重升的操作策略：**

起步阶段的核心策略是"小仓试错，确认后加仓"。具体要点如下：

第一，小仓试探。确认是新题材启动后，以不超过20%的仓位试探性买入。涅盘重升说："起步期不确定性很大，一定要控制仓位。"

第二，分批建仓。如果股票继续上涨，确认龙头地位后，逐步加仓。

第三，不贪心。在起步阶段，龙头股的地位尚未完全确认，不要盲目重仓。

第四，设定止损。设定严格的止损位，如果股票走势不符合预期，及时止损出局。

### 5.2.3 加速阶段（主升）

加速阶段是情绪周期中最甜蜜的时期，是主升浪发生的时间。这个阶段的市场特征是：龙头股持续涨停，板块联动性强，赚钱效应显著，资金积极进场。

**涅盘重升的主升浪理论：**

涅盘重升认为，主升浪是情绪周期中盈利的黄金期。他说："主升浪是市场中最好赚钱的时期，只要买对了股票，闭着眼睛都能赚钱。但主升浪也是最容易亏钱的时期，因为很多人会在主升浪中被震出去。"

主升浪的核心特征是"趋势加速"。涅盘重升解释道："趋势加速意味着股票进入快速上涨阶段，每天的涨幅都很大，但同时波动也会加剧。很多人受不了波动，在趋势加速的过程中被洗出去。"

**加速阶段的市场特征：**

第一，龙头股持续涨停。龙头股从二板、三板一路涨到五板、六板甚至更高，连板记录不断刷新。

第二，板块联动增强。龙头股的赚钱效应扩散到整个板块，跟风股、补涨股纷纷出现，板块形成明显的梯队结构。

第三，资金积极进场。赚钱效应吸引场外资金加速进场，成交量持续放大，推动行情进一步发展。

第四，涨停数量增加。从起步期的10-30只增加到30-50只，市场赚钱效应显著。

第五，连板高度提升。市场出现4板、5板甚至更高的连板股票，连板高度不断提升。

第六，炸板率下降。从起步期的40-60%下降到20-40%，大多数涨停股票能够封住涨停。

**加速阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | 30-50 | 每日涨停30-50只 |
| 跌停数 | 10-20 | 每日跌停10-20只 |
| 连板数 | 5-10 | 最高连板为5-10板 |
| 涨跌比 | 1-2 | 上涨股票数量明显多于下跌 |
| 炸板率 | 20-40% | 炸板率明显下降 |
| 成交额 | 较起步期放大50%以上 | 市场活跃度显著提升 |

**加速阶段的核心特征：主升浪**

加速阶段最重要的特征是主升浪的出现。主升浪是股票上涨过程中最快、最猛的一段，通常持续5-10个交易日，涨幅可达50%-100%甚至更高。

涅盘重升强调，主升浪的持有策略是"持有为主，不轻易卖出"。他说："主升浪中最好的策略就是持有盈利的股票，享受上涨带来的利润。不要试图做T，不要试图高抛低吸，因为你的操作可能会让你失去仓位。"

在主升浪阶段，应该坚持以下原则：

第一，持有为主。在主升浪阶段，最好的策略是持有盈利的股票，享受上涨带来的利润，不要轻易卖出。

第二，回调加仓。如果股票出现回调，在回调缩量的位置加仓，增加盈利的筹码。

第三，不轻易言顶。在主升浪阶段，不要轻易猜测顶部，只要上涨趋势没有改变，就继续持有。

第四，趋势为王。只要均线系统多头排列，MACD红柱持续放大，就继续持有，不因短期波动而卖出。

### 5.2.4 高潮阶段（鼎盛）

高潮阶段是情绪周期的巅峰时期，是赚钱效应达到顶峰的阶段。这个阶段的市场特征是：市场情绪达到沸点，全民炒股情绪高涨，龙头股冲击高位，涨停数量创历史新高。

**涅盘重升的高潮警示：**

涅盘重升特别强调，高潮期是最危险的时期。他说："高潮期是散户进场的高峰，也是专业投资者撤退的时期。当所有人都在谈论股票的时候，往往是市场即将见顶的信号。"

高潮期的风险包括：

"第一，龙头股随时可能炸板。在高潮期，龙头股已经涨幅巨大，随时可能出现炸板，一旦炸板未能回封，可能引发连锁反应。第二，获利盘巨大。持仓者的盈利已经非常丰厚，随时可能兑现利润，一旦抛压增加，股价可能快速下跌。第三，情绪过于亢奋。当所有人都在谈论股票时，往往是市场即将见顶的信号。第四，成交量过大。成交额创出天量时，往往意味着资金已经开始出逃。"

**高潮阶段的市场特征：**

第一，涨停数量创纪录。每日涨停股票数量可能超过100只，甚至达到150只以上，市场极度活跃。

第二，龙头股冲击高位。龙头股连板高度不断刷新，从6板、7板一直涨到10板以上，向历史记录发起冲击。

第三，全民炒股情绪。街头巷尾都在讨论股票，朋友圈充满炒股赚钱的消息，新开户数量大增。

第四，场外资金跑步进场。赚钱效应吸引场外资金加速进场，成交量创出阶段性新高。

第五，涨跌比极高。上涨股票数量远多于下跌股票数量，涨跌比可能超过2，甚至达到3-4。

第六，炸板率极低。大多数涨停股票能够封住涨停，炸板率可能低于20%，追涨资金都能获得盈利。

**高潮阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | > 100 | 每日涨停超过100只 |
| 跌停数 | < 10 | 每日跌停少于10只 |
| 连板数 | > 10 | 最高连板超过10板 |
| 涨跌比 | > 2 | 上涨股票数量是下跌的2倍以上 |
| 炸板率 | < 20% | 炸板率极低 |
| 成交额 | 创近期峰值 | 成交金额达到阶段性高点 |

**高潮阶段的风险提示：盛极而衰**

高潮阶段虽然赚钱效应显著，但也是风险积聚的时期。经验丰富的投资者都知道，股市有句老话："行情在绝望中产生，在犹豫中上涨，在欢乐中死亡。"高潮阶段正是"欢乐中死亡"的时期。

**涅盘重升的操作策略：**

高潮阶段的核心策略是"分批止盈，不贪心"。具体要点如下：

第一，分批止盈。涅盘重升说："高潮期能够获得50%-100%的收益已经非常可观，不要追求卖在最高点。在高位出现滞涨信号时就要开始分批止盈。"

第二，不贪心。涅盘重升特别强调不贪心，他说："贪心是散户亏钱的最大原因之一。在高潮期，能够及时锁定利润的人才是真正的赢家。"

第三，准备退潮。高潮阶段随时可能结束，要在心理上和操作上做好准备，一旦出现退潮信号，立即清仓。

第四，锁定利润。把账户利润变成真正的现金，不要只是账面盈利。

### 5.2.5 发散阶段（分歧）

发散阶段是情绪周期从高潮转向退潮的过渡期，是市场开始出现分歧的时期。这个阶段的市场特征是：龙头股在高位出现分歧，板块轮动加快，题材开始扩散。

**涅盘重升的分歧识别：**

涅盘重升认为，发散阶段是情绪周期中比较微妙的阶段。他说："发散阶段的特点是'看着很强，但已经不强了'。很多人还在追高，但真正的高手已经开始撤退。"

发散阶段的核心特征是"分歧"。涅盘重升解释道："分歧体现在几个方面：第一，龙头股在高位出现分歧，有人继续看多，有人开始看空。第二，板块内部出现分化，有的股票继续涨，有的股票开始跌。第三，成交量虽然很大，但涨跌股票数量开始趋于平衡。"

**发散阶段的市场特征：**

第一，龙头股高位分歧。龙头股在高位无法继续封住涨停，出现炸板或高位震荡，龙头股的上涨动力开始减弱。

第二，板块轮动加快。热点不再集中在一个板块，而是在多个板块间快速轮动，一天可能出现多个热点，但没有持续性。

第三，题材扩散。龙头股的题材开始向相关板块扩散，但扩散后的跟风股力度明显减弱。

第四，涨停数量维持高档。虽然涨停数量仍然较多，但质量开始下降，封板质量不如高潮期。

第五，炸板率上升。从高潮期的低于20%上升到20-40%，涨停股票打开的概率增加。

第六，亏钱效应开始出现。部分高位股开始下跌，跟风股出现补跌，亏钱效应开始扩散。

**发散阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | 50-100 | 每日涨停50-100只 |
| 跌停数 | 20-30 | 每日跌停20-30只 |
| 连板数 | 5-10 | 最高连板为5-10板 |
| 涨跌比 | 1-2 | 上涨股票数量多于下跌 |
| 炸板率 | 20-40% | 炸板率开始上升 |
| 成交额 | 维持在高位但不创新高 | 成交金额停止增长 |

**发散阶段的核心任务：识别新龙头和切换方向**

发散阶段最重要的任务是观察是否有机会切换到新龙头，或者是否需要减仓回避风险。

第一，观察是否有新题材出现。涅盘重升说："如果在发散期有新的题材出现，并形成新的龙头，可能带领市场继续上攻。如果没有新题材，则市场可能进入退潮期。"

第二，观察龙头股的走势。如果龙头股高位炸板后能够继续封板，可能是继续上涨的信号。如果龙头股高位炸板后无法回封，则可能是退潮的开始。

第三，观察板块轮动情况。如果板块轮动有序，健康轮动，可能维持震荡。如果板块轮动混乱，快速切换，则可能是退潮的前兆。

**涅盘重升的操作策略：**

发散阶段的核心策略是"切换新题材，不恋战"。具体要点如下：

第一，不恋战。如果手中持有的是高位股，在发散阶段应该逐步减仓，不要期待更高。

第二，切换新题材。如果有新题材出现，可以将仓位切换到新题材的新龙头。

第三，快进快出。在发散阶段，操作难度加大，应该快进快出，不宜恋战。

第四，准备退潮。如果市场没有新题材，热点开始消退，应该做好准备，随时可能进入退潮期。

### 5.2.6 退潮阶段（亏钱）

退潮阶段是情绪周期中亏钱效应开始扩散的时期，是市场情绪从盛转衰的阶段。这个阶段的市场特征是：龙头股断板，板块分化严重，资金撤离。

**涅盘重升的退潮警示：**

退潮阶段最危险的特征是踩踏出逃。涅盘重升说："退潮期最怕的就是踩踏。当市场开始转弱时，持有股票的投资者开始争相卖出，导致股价快速下跌，形成踩踏效应。"

在退潮阶段，常见的踩踏场景包括：

"第一，龙头股高位炸板后持续下跌。龙头股在高位炸板后，无法反包，一路下跌，持有者争相出逃。第二，跟风股大幅低开。前一天涨停的跟风股第二天大幅低开，持有者亏损出局。第三，强势股补跌。前期强势的股票在退潮期开始补跌，跌幅可能超过30%。"

**退潮阶段的市场特征：**

第一，龙头股断板。龙头股无法继续封住涨停，高位股纷纷炸板，龙头股的连板记录终结。

第二，板块分化严重。跟风股纷纷下跌，部分跟涨的股票甚至出现跌停，板块内部分化严重。

第三，资金撤离。获利资金开始撤离市场，成交量虽然可能仍然较大，但主要是出货所致。

第四，涨停数量减少。从高潮期的100只以上减少到50只以下，市场活跃度下降。

第五，跌停数量增加。从高潮期的不足10只增加到30只以上，亏钱效应明显。

第六，炸板率上升。从发散期的20-40%上升到40%以上，涨停股票打开的概率大增。

**退潮阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | < 50 | 每日涨停不足50只 |
| 跌停数 | > 30 | 每日跌停超过30只 |
| 连板数 | < 5 | 最高连板不足5板 |
| 涨跌比 | < 1 | 上涨股票数量少于下跌 |
| 炸板率 | > 40% | 炸板率明显上升 |
| 成交额 | 开始萎缩 | 成交金额开始下降 |

**退潮阶段操作策略：**

这个阶段的核心策略是"清仓观望，不抄底"。具体要点如下：

第一，及时清仓。在退潮阶段，应该果断清仓，不要持有股票过夜。

第二，不抄底。涅盘重升特别强调不抄底，他说："退潮期的股票可能看着很便宜，但不知道底部在哪里。盲目抄底可能继续被套。"

第三，等待新周期。退潮阶段是旧周期的结束，应该耐心等待新周期的启动。

第四，控制风险。退潮阶段的亏损速度很快，要严格控制风险，避免大幅回撤。

### 5.2.7 淘汰阶段（冰点前兆）

淘汰阶段是情绪周期中最惨烈的时期，是高位股大跌、跟风股补跌、市场恐慌的阶段。这个阶段是退潮期的高峰，也是新周期酝酿的前兆。

**涅盘重升的淘汰期认知：**

淘汰阶段是情绪周期中最难熬的时期。涅盘重升说："淘汰期最大的特点就是'多杀多'，高位股持有者互相踩踏，股价持续暴跌。这个时期最重要的是保存实力，不要想着抄底，不要想着抢反弹。"

淘汰阶段和酝酿阶段都表现出市场极度低迷的特征，但两者有重要区别：

"第一，位置不同。淘汰阶段是高位股从高处跌落，跌幅巨大；酝酿阶段是股价已经跌无可跌，处于低位。第二，性质不同。淘汰阶段是风险的释放，是下跌的延续；酝酿阶段是机会的酝酿，是上涨的前兆。第三，操作不同。淘汰阶段仍然不能抄底，要等待；酝酿阶段可以开始关注，寻求试错机会。"

**淘汰阶段的市场特征：**

第一，高位股大跌。前期涨幅巨大的龙头股、跟风股纷纷大幅下跌，部分股票可能连续跌停。

第二，跟风股补跌。前期相对抗跌的跟风股开始补跌，跌幅可能超过高位股。

第三，市场恐慌。投资者恐慌情绪蔓延，争相卖出股票，导致股价持续下跌。

第四，成交量可能放大。在恐慌初期，可能有放量的杀跌盘出逃，导致成交量放大。

第五，涨停数量极少。市场几乎没有什么亮点，涨停股票数量可能不足20只。

第六，跌停数量众多。跌停股票数量可能超过50只，部分股票连续跌停。

**淘汰阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | < 20 | 每日涨停不足20只 |
| 跌停数 | > 50 | 每日跌停超过50只 |
| 连板数 | < 3 | 最高连板不足3板 |
| 涨跌比 | < 0.5 | 下跌股票数量远超上涨 |
| 炸板率 | > 60% | 炸板率极高 |
| 成交额 | 较高峰明显萎缩 | 市场成交清淡 |

**淘汰阶段操作策略：**

这个阶段的核心策略是"空仓观望，不操作"。具体要点如下：

第一，空仓观望。这个阶段最重要的是保存实力，不做任何操作。

第二，不抄底。淘汰阶段的股票虽然跌幅巨大，但不知道底部在哪里，盲目抄底可能继续被套。

第三，不抢反弹。抢反弹是短线操作中最危险的行为之一，尤其是在淘汰阶段。

第四，等待冰点。淘汰阶段之后就是情绪冰点，应该等待冰点出现后再关注机会。

### 5.2.8 新周期（回到起点）

新周期阶段是淘汰阶段之后的重置阶段，是市场从冰点重新启动的时期。这个阶段的市场特征与起步阶段相似，但可能伴随新的题材和新的龙头。

**涅盘重升的新周期判断：**

新周期阶段是情绪周期重新启动的时期。涅盘重升说："新周期和起步期很像，但不完全一样。新周期是从极度低迷中起步，起步期是从正常状态起步。两者的区别在于，新周期的底部更加扎实。"

新周期和起步期的主要区别在于：

"第一，起点不同。新周期是从情绪冰点起步，情绪极度低迷；起步阶段是从回暖起步，情绪已经转暖。第二，风险不同。新周期的风险是可能再次退潮，底部可能不止一个；起步阶段的风险是题材可能不够大，持续性存疑。第三，策略微调。新周期应该更加谨慎，仓位更低一些；起步阶段可以适度提高仓位。"

**新周期阶段的市场特征：**

第一，情绪冰点过后。经历了淘汰阶段的惨烈下跌后，市场情绪达到极致冰点，然后开始缓慢回暖。

第二，新题材出现。可能有新的题材出现，这些题材可能是全新的题材，或者是老题材的复苏。

第三，龙头股启动。某个股票率先涨停，成为新周期的龙头股，拉开新周期上涨的序幕。

第四，资金重新入场。先知先觉的资金开始试探性买入，市场成交量开始温和放大。

**新周期阶段的识别量化指标：**

| 指标 | 量化标准 | 说明 |
|------|----------|------|
| 涨停数 | 10-30 | 每日涨停10-30只 |
| 跌停数 | 20-50 | 每日跌停20-50只 |
| 连板数 | 3-5 | 最高连板为3-5板 |
| 涨跌比 | 0.5-1 | 上涨与下跌股票数量基本持平 |
| 炸板率 | 40-60% | 炸板率有所下降 |
| 成交额 | 开始温和放大 | 市场活跃度开始提升 |

**新周期阶段操作策略：**

这个阶段的核心策略与起步阶段类似："小仓试错，确认后加仓"。具体要点如下：

第一，小仓试错。新周期的开始阶段，应该以不超过20%的仓位试探性买入。

第二，确认后加仓。如果股票能够继续上涨，确认龙头地位后，逐步加仓。

第三，不贪心。新周期的持续性尚未确认，不要盲目重仓。

第四，设定止损。设定严格的止损位，如果走势不符预期，及时止损。

### 5.3.2 冰点判断标准

情绪冰点是情绪周期中最重要的节点之一，是逆向布局的重要时机。以下是冰点的量化判断标准：

**冰点的核心指标：**

第一，涨停数 < 10。每日涨停股票数量不足10只，市场几乎没有什么亮点。

第二，跌停数 > 50。每日跌停股票数量超过50只，亏钱效应显著。

第三，涨跌比 < 0.5。上涨股票数量远少于下跌股票数量，市场呈现普跌格局。

第四，连板数 < 3。最高连板高度不超过3板，连板股票非常稀少。

第五，成交额较高峰萎缩50%以上。市场参与度极低，成交量严重萎缩。

**冰点的辅助判断：**

第一，高位股持续下跌。前期涨幅巨大的龙头股、跟风股持续下跌，跌幅已经很大。

第二，市场情绪极度悲观。投资者对市场前景极度悲观，悲观情绪弥漫。

第三，媒体和社交圈不再讨论股票。街头巷尾不再讨论股票，股票话题从生活中消失。

第四，身边投资者普遍亏损。大多数投资者都处于亏损状态，亏损幅度较大。

**涅盘重升的冰点识别法：**

"我识别冰点，主要看三个信号：第一，涨停数量是否达到极低水平；第二，跌停数量是否达到极高水平；第三，市场是否出现极度悲观情绪。当这三个信号同时出现时，往往就是冰点。"

**冰点的意义：**

情绪冰点意味着市场恐慌情绪已经充分释放，可能砸出股票的合理价格甚至低估价格。同时，冰点也是新周期的孕育期，预示着新的机会即将出现。

### 5.3.3 高潮判断标准

情绪高潮是情绪周期中最危险的时间点，是卖出的最佳时机。以下是高潮的量化判断标准：

**高潮的核心指标：**

第一，涨停数 > 100。每日涨停股票数量超过100只，甚至达到150只以上。

第二，涨跌比 > 2。上涨股票数量是下跌股票数量的2倍以上，市场呈现普涨格局。

第三，连板数 > 10。最高连板高度超过10板，龙头股连板记录不断刷新。

第四，炸板率 < 20%。大多数涨停股票能够封住涨停，追涨者都能获利。

第五，成交额创近期峰值。成交金额达到阶段性高点，可能接近历史天量。

**高潮的辅助判断：**

第一，全民讨论股票。街头巷尾都在讨论股票，朋友圈充满炒股赚钱的消息。

第二，新开户数量大增。投资者跑步入场，新开户数量创历史新高或阶段性新高。

第三，媒体大量报道。新闻媒体大量报道股市上涨，投资者信心爆棚。

第四，股神遍地走。到处都在传有人炒股暴富的故事，似乎炒股赚钱很容易。

**涅盘重升的高潮识别法：**

"我识别高潮，主要看三个信号：第一，涨停数量是否达到极高水平；第二，市场是否出现全民炒股的狂热情绪；第三，成交量是否创出天量。当这三个信号同时出现时，往往就是高潮。"

**高潮的意义：**

情绪高潮意味着市场赚钱效应已经达到顶峰，可能很快就会盛极而衰。同时，高潮也是锁定利润的最佳时机，可以在高潮时分批卖出，锁定盈利。

## 5.4 情绪周期实战案例分析

### 5.4.1 案例一：完整情绪周期（2026年1月-3月人工智能题材）

**周期概述：**

2026年1月至3月，人工智能题材走出了一波完整的情绪周期，从冰点到高潮再到退潮，典型地展示了情绪周期的各个阶段。

**第一阶段：酝酿期（1月1日-1月10日）**

市场特征分析：

1月份上旬，市场处于上一波行情退潮后的低迷期。涨停数量极少，每日涨停不足10只，最低时仅有5只。跌停数量众多，每日跌停超过50只，部分高位股持续下跌。成交额较高峰期萎缩超过50%，市场冷清，投资者情绪悲观。

操作记录：

这个阶段保持空仓观望，不进行任何操作。耐心等待市场情绪转暖的信号。

**第二阶段：起步期（1月11日-1月20日）**

市场特征分析：

1月中旬，人工智能相关消息开始发酵，某科技股率先涨停，成为人工智能题材的龙头股。涨停数量从不足10只增加到20只左右，跌停数量从超过50只减少到30只左右。市场开始出现新题材的苗头。

操作记录：

1月15日，在龙头股首板时以小仓15%试探性买入。设置严格的止损位，如果走势不符预期，亏损不超过5%时止损出局。

**第三阶段：加速期（1月21日-2月5日）**

市场特征分析：

龙头股从二板连续涨停到五板，连板高度不断提升。人工智能板块内多只股票涨停，板块联动效应明显。涨停数量从20只增加到50只左右，跌停数量减少到10只左右。成交量持续放大，场外资金加速进场。

操作记录：

1月22日，龙头股二板确认，以50%仓位加仓。1月25日，龙头股回调后缩量，再次加仓至80%。1月28日，龙头股继续涨停，仓位保持80%，享受主升浪的利润。

**第四阶段：高潮期（2月6日-2月15日）**

市场特征分析：

人工智能题材成为市场最热门的话题，涨停数量连续多日超过100只，最高达130只。龙头股冲击十连板，创下年内连板记录。全民讨论股票，场外资金跑步进场。成交额创出年内新高。

操作记录：

2月8日，龙头股高位震荡，出现第一次炸板。在炸板后以85元的价格卖出50%仓位，锁定部分利润。2月10日，龙头股继续上涨，但成交量开始萎缩，以90元的价格卖出剩余50%仓位，清仓离场。综合收益约为120%。

**第五阶段：退潮期（2月16日-2月25日）**

市场特征分析：

龙头股在高位无法继续封板，开始连续下跌。跟风股纷纷下跌，部分股票出现跌停。涨停数量从100只以上减少到不足50只，跌停数量增加到30只以上。资金开始撤离市场。

操作记录：

这个阶段保持空仓，不参与退潮期的反弹操作。因为知道退潮期的风险，远远大于可能的收益。

**第六阶段：淘汰期（2月26日-3月5日）**

市场特征分析：

高位股大幅下跌，部分股票连续跌停。跟风股纷纷补跌，市场恐慌情绪蔓延。涨停数量不足20只，跌停数量超过50只。投资者情绪极度悲观。

操作记录：

这个阶段保持空仓观望，耐心等待情绪冰点的出现和新周期的启动信号。

**第七阶段：新周期（3月6日起）**

市场特征分析：

3月上旬，市场情绪开始缓慢回暖。可能出现新的题材，龙头股开始启动。市场进入新的情绪周期。

操作记录：

根据新题材的出现，以小仓开始试探新的周期。

### 5.4.2 案例二：快速情绪周期（2026年4月新能源题材）

**周期概述：**

2026年4月，新能源题材走出了一波快速的情绪周期，持续时间短，涨跌速度快，对操作的要求更高。

**周期特征分析：**

这次快速周期呈现以下特点：

第一，酝酿期短。4月初的酝酿期仅有3天，比正常周期缩短了一半以上。

第二，上涨速度快。从起步到加速到高潮仅用了5天时间，龙头股连板速度极快。

第三，下跌速度也快。从高潮到退潮到淘汰仅用了3天时间，高位股连续跌停。

第四，整体周期短。整个周期仅持续约10天，是一次"短平快"的情绪周期。

**涅盘重升的快周期应对策略：**

面对快速情绪周期，涅盘重升的操作要点是：

"第一，快进快出。因为周期短，持有时间不能太长，必须快进快出。第二，不贪心。因为上涨时间短，收益预期不能太高，达到目标就要卖出。第三，严止损。因为下跌速度快，一旦走势不符预期，必须立即止损，不能犹豫。"

**操作复盘：**

4月6日，新能源题材龙头股首板。以20%仓位试探买入。4月7日，龙头股二板，以50%仓位加仓。4月8日，龙头股三板，仓位保持70%。4月9日，龙头股高位炸板，在炸板后以收盘价卖出，锁定30%收益。

4月10日-12日，市场进入退潮期，保持空仓，没有参与。4月13日，市场进入淘汰期，高位股连续跌停，保持空仓观望。

**经验教训：**

面对快速情绪周期，关键是要识别周期的速度。如果判断是快速周期，就要相应调整操作策略：降低收益预期，缩短持有时间，严格止损。否则，可能会错过最佳卖出时机，或者在退潮期被套。

### 5.4.3 案例三：双峰结构（2026年5月芯片题材）

**周期概述：**

2026年5月，芯片题材走出了一波双峰结构的情绪周期，即在第一次高潮后，经过短暂调整，再次启动第二波行情。

**周期特征分析：**

这次双峰周期呈现以下特点：

第一，第一次高潮时，龙头股达到8连板。5月初，芯片题材启动，龙头股连续涨停，在第一次高潮时达到8连板。

第二，第一次退潮时，跌幅相对有限。龙头股从8连板的高位回调，但回调幅度约30%，没有完全崩溃。

第三，短暂整理后，再次启动。回调20个交易日后，芯片题材再次启动，龙头股再次涨停，形成第二峰。

第四，第二峰高度低于第一峰。第二次高潮时，龙头股仅达到6连板，高度低于第一峰。

**涅盘重升的双峰操作策略：**

面对双峰结构，涅盘重升的操作要点是：

"第一，第一次高潮时分批止盈。在第一次高潮时，应该分批卖出，锁定大部分利润。第二，保持关注，等待第二峰。如果第一次高潮后跌幅有限，板块没有完全退潮，可以保持关注，等待第二峰的机会。第三，仓位要轻，因为是二次操作。在等待第二峰时，仓位要相对轻一些，因为不确定性增加。第四，第二峰要及时出逃。第二峰的高度通常低于第一峰，所以要及时止盈，不能恋战。"

**操作复盘：**

5月3日，芯片题材龙头股首板，以30%仓位试探买入。5月5日，龙头股三板，以70%仓位加仓。5月8日，龙头股六板，仓位保持80%。5月10日，龙头股高位炸板，以涨停价卖出，锁定80%收益。

5月11日-30日，市场进入退潮期和整理期。保持空仓观望，关注芯片板块的动态。

5月31日，芯片板块再次启动，新龙头出现。以25%仓位试探买入。6月2日，新龙头二板，以50%仓位加仓。6月4日，新龙头四板，以65%仓位持有。6月5日，新龙头高位震荡，以收盘价卖出，锁定45%收益。

**经验教训：**

双峰结构是情绪周期中较难把握的形态，需要准确判断板块是否还有第二波的动力。关键是要观察板块的整体状态，如果板块龙头在第一峰后没有完全崩溃，板块内仍有其他股票活跃，那么可能出现第二峰。如果板块龙头在第一峰后持续低迷，板块整体退潮，则可能没有第二峰。

## 5.5 情绪周期各阶段的实战运用

### 5.5.1 酝酿期实战运用

**操作策略：空仓等待，小仓试错**

酝酿期是情绪周期中最难操作的时期，因为市场机会稀少，亏钱效应明显。正确的做法是保持空仓，保存实力，等待机会。

**涅盘重升的酝酿期建议：**

"酝酿期最重要的是保持耐心，不要想着抄底，不要想着买在最低点。这个时期应该多看少动，等待机会。"

**实战要点：**

第一，观察新题材的苗头。酝酿期虽然机会少，但可以开始关注市场，观察是否有新题材出现的苗头。

第二，等待标志性信号。当市场出现连续涨停的股票，或者成交金额开始放大，可能是新周期即将启动的信号。

第三，不盲目抄底。即使股价已经跌幅很大，也不要盲目抄底，因为可能还有更低。

第四，保持耐心。酝酿期可能持续较长时间，保持耐心，等待机会。

**仓位管理：**

酝酿期的仓位应该控制在10%以下，以极小的仓位试错，或者直接空仓。

**止损策略：**

如果在小仓试错后走势不符预期，应该立即止损出局，不要恋战。

### 5.5.2 起步期实战运用

**操作策略：小仓试错，确认后加仓**

起步期是新周期的启动阶段，操作的核心是"小仓试错，确认后加仓"，即在确认新周期启动后再逐步加仓。

**涅盘重升的起步期建议：**

"起步期是布局的时期，但不确定性很大。这个时期要以小仓位试探，确认后再逐步加仓。"

**实战要点：**

第一，识别新题材。新题材必须具备新颖性、想象空间、事件驱动等特点。

第二，找到龙头股。龙头股通常是第一个涨停的股票，具有先发优势。

第三，观察板块联动。龙头股涨停后，同板块的股票是否跟涨，联动性是否强。

第四，观察成交量。龙头股的成交量必须明显放大，表明有资金积极参与。

**仓位管理：**

起步期的仓位应该控制在20%-30%，确认龙头后再逐步加仓。

**止损策略：**

设定严格的止损位，通常设定在买入价的-5%到-7%。如果走势不符预期，触及止损位，立即止损。

### 5.5.3 加速期实战运用

**操作策略：主升段持有，回调缩量加仓**

加速期是情绪周期中最甜蜜的时期，是盈利的黄金期。操作的核心是"主升段持有，回调缩量加仓"，即持有盈利的股票，享受主升浪的利润。

**涅盘重升的加速期建议：**

"加速期是赚钱最容易的时期，这个时期最好的策略就是持有。持有盈利的股票，不要轻易卖出。"

**实战要点：**

第一，持有为主。在加速期，不要轻易卖出，只要上涨趋势没有改变，就继续持有。

第二，回调加仓。如果股票出现回调，在回调缩量的位置加仓，增加盈利的筹码。

第三，不轻易言顶。在加速期，不要猜测顶部，只要均线多头排列，就继续持有。

第四，关注龙头地位。如果持有的是龙头股，只要龙头的地位没有改变，就继续持有。

**仓位管理：**

加速期的仓位可以提升到80%以上，享受主升浪的利润。

**止盈策略：**

可以设定移动止盈位，例如从最高点回落10%时止盈，或者均线死叉时止盈。

### 5.5.4 高潮期实战运用

**操作策略：分批止盈，不贪心**

高潮期是情绪周期中最危险的时间点，虽然赚钱效应仍然显著，但风险正在积聚。操作的核心是"分批止盈，不贪心"，即在高潮时分批卖出，锁定利润。

**涅盘重升的高潮期建议：**

"高潮期是最危险的时期，这个时期要时刻保持警惕。一旦出现滞涨信号，就要考虑卖出。"

**实战要点：**

第一，分批止盈。不要等到最高点才卖，在高位出现滞涨信号时就开始分批止盈。

第二，不贪心。在高潮期，能够获得50%-100%的收益已经非常可观，不要追求卖在最高点。

第三，观察龙头信号。如果龙头股在高位持续无法封板，或者炸板后无法回封，可能是退潮的信号。

第四，准备撤退。一旦出现退潮信号，立即清仓，不要恋战。

**仓位管理：**

高潮期的仓位应该逐步降低，从80%以上降低到50%，再降低到20%，最后清仓。

**止盈策略：**

设定明确的止盈目标，例如从买入价上涨80%时卖出一半，上涨100%时卖出剩余的一半。

### 5.5.5 发散期实战运用

**操作策略：切换新题材，不恋战**

发散期是情绪周期中充满机会但也充满风险的时期。操作的核心是"切换新题材，不恋战"，即如果发现新题材，可以切换，否则应该减仓。

**涅盘重升的发散期建议：**

"发散期是一个过渡期，这个时期要观察是否有新的机会出现。如果没有新机会，就要考虑撤退。"

**实战要点：**

第一，观察是否有新龙头。如果有新的题材出现并形成新的龙头，可以将仓位切换到新龙头上。

第二，不恋战。如果手中持有的是高位股，在发散期应该逐步减仓，不要期待更高。

第三，快进快出。在发散期，操作难度加大，应该快进快出，不宜恋战。

第四，准备退潮。如果市场没有新题材，热点开始消退，应该做好准备，随时可能进入退潮期。

**仓位管理：**

发散期的仓位应该控制在50%以下，如果发现新龙头可以适当提高，但不宜重仓。

**止损策略：**

如果持有的是高位股，一旦出现高位滞涨或者炸板，应该立即止损。

### 5.5.6 退潮期实战运用

**操作策略：清仓观望，不抄底**

退潮期是情绪周期中亏钱效应开始扩散的时期。操作的核心是"清仓观望，不抄底"，即及时卖出持股，空仓等待。

**涅盘重升的退潮期建议：**

"退潮期是最容易亏钱的时期，这个时期最好的策略就是休息。不要想着抄底，不要想着抢反弹。"

**实战要点：**

第一，及时清仓。在退潮期，应该果断清仓，不要持有股票过夜。

第二，不抄底。退潮期的股票可能看着很便宜，但不知道底部在哪里，盲目抄底可能继续被套。

第三，等待新周期。退潮期是旧周期的结束，应该耐心等待新周期的启动。

第四，控制风险。退潮期的亏损速度很快，要严格控制风险，避免大幅回撤。

**仓位管理：**

退潮期的仓位应该控制在0%，保持空仓。

**止损策略：**

如果在退潮期还持有股票，一旦触及止损位，立即止损，不要犹豫。

### 5.5.7 淘汰期实战运用

**操作策略：空仓观望，不操作**

淘汰期是情绪周期中最惨烈的时期。操作的核心是"空仓观望，不操作"，即不进行任何操作，耐心等待。

**涅盘重升的淘汰期建议：**

"淘汰期是情绪周期中最难熬的时期，这个时期最重要的是保存实力。不要想着抄底，不要想着抢反弹，耐心等待冰点的出现。"

**实战要点：**

第一，空仓观望。这个阶段最重要的是保存实力，不做任何操作。

第二，不抄底。淘汰阶段的股票虽然跌幅巨大，但不知道底部在哪里，盲目抄底可能继续被套。

第三，不抢反弹。抢反弹是短线操作中最危险的行为之一，尤其是在淘汰期。

第四，等待冰点。淘汰期之后就是情绪冰点，应该等待冰点出现后再关注机会。

**仓位管理：**

淘汰期的仓位应该控制在0%，保持空仓。

**止损策略：**

如果在淘汰期还持有股票，应该不计成本地卖出，保存实力。

### 5.5.8 新周期实战运用

**操作策略：小仓试错，确认后加仓**

新周期是情绪周期的重新启动阶段，操作策略与起步期类似："小仓试错，确认后加仓"。

**涅盘重升的新周期建议：**

"新周期是重新布局的时期，但要比起步期更加谨慎。因为刚经历过淘汰期，市场信心还没有完全恢复。"

**实战要点：**

第一，小仓试错。新周期的开始阶段，应该以不超过20%的仓位试探性买入。

第二，确认后加仓。如果股票能够继续上涨，确认龙头地位后，逐步加仓。

第三，不贪心。新周期的持续性尚未确认，不要盲目重仓。

第四，设定止损。设定严格的止损位，如果走势不符预期，及时止损。

**仓位管理：**

新周期初期仓位应该控制在20%-30%，确认后逐步加仓到50%-70%。

**止损策略：**

设定严格的止损位，通常设定在买入价的-7%到-10%。如果走势不符预期，触及止损位，立即止损。

## 5.6 情绪周期与大盘环境的配合

### 5.6.1 大盘环境与情绪周期的关系

大盘环境和情绪周期是相互关联的两个维度。大盘环境反映的是指数和整体市场的状态，情绪周期反映的是市场赚钱效应和短线活跃度。两者结合可以更准确地判断市场状态。

**涅盘重升的综合研判法：**

涅盘重升认为，大盘环境和情绪周期应该结合起来分析。他说："单独看大盘环境，可能错过很多机会；单独看情绪周期，可能看不清全局。最好的方法是两者结合，综合研判。"

**强势大盘环境下的情绪周期：**

当大盘环境处于强势时，情绪周期各阶段可能呈现以下特点：

- 酝酿期可能很短，强势环境下的冰点往往很快被打破
- 起步期和加速期可能更加猛烈，龙头股连板高度可能更高
- 高潮期可能持续更长时间，强势环境下赚钱效应更容易维持
- 退潮期可能相对温和，强势环境下回调幅度可能较小

**弱势大盘环境下的情绪周期：**

当大盘环境处于弱势时，情绪周期各阶段可能呈现以下特点：

- 酝酿期可能很长，弱势环境下的冰点可能持续较长时间
- 起步期和加速期可能力度较弱，龙头股连板高度有限
- 高潮期可能很短暂，弱势环境下赚钱效应难以维持
- 退潮期可能更加惨烈，弱势环境下回调幅度可能更大

### 5.6.2 择时系统的综合运用

在实际操作中，应该综合考虑大盘环境和情绪周期，制定操作策略。

**强势大盘环境 + 情绪周期加速期/高潮期：**

这是最好的操作时机，应该满仓操作，持有龙头股，享受主升浪的利润。

**强势大盘环境 + 情绪周期起步期/发散期：**

这是较好的操作时机，可以小仓参与试错，或者切换到新龙头上。

**弱势大盘环境 + 情绪周期退潮期/淘汰期：**

这是最差的操作时机，应该保持空仓，不进行任何操作。

**弱势大盘环境 + 情绪周期酝酿期/起步期：**

这是观望为主的时机，可以小仓试探，但严格控制仓位和止损。

**平衡大盘环境 + 情绪周期各阶段：**

这是需要谨慎操作的时机，仓位应该相对保守，快进快出。

**涅盘重升的择时建议：**

"择时是短线操作的核心。我的择时系统是：先看大盘环境，再看情绪周期。大盘环境决定了可以不可以做，情绪周期决定了做什么。两者结合，才能制定完整的交易计划。"

## 5.7 本章小结

情绪周期是股票市场运行的基本规律，理解它并正确运用它，是短线交易成功的关键。

**核心要点回顾：**

第一，情绪周期分为8个阶段：酝酿期、起步期、加速期、高潮期、发散期、退潮期、淘汰期、新周期。周而复始，循环往复。

第二，不同的情绪周期阶段需要采用不同的操作策略：高潮期满仓持有，冰点期空仓等待。

第三，判断情绪周期需要综合考虑多个量化指标：涨停数、跌停数、连板数、涨跌比、炸板率等。

第四，情绪周期与大盘环境应该配合使用，强势大盘环境下可以更激进，弱势大盘环境下应该更保守。

第五，避免在情绪周期的运用中犯以下错误：高潮时贪婪、冰点时恐惧、混淆不同阶段、忽视风险控制。

**涅盘重升的箴言：**

> "情绪周期就像潮汐，有涨潮、有退潮、有平潮。理解了这个规律，就能在正确的时机做正确的事情。"

> "高潮期是散户进场的高峰，也是专业投资者撤退的时期。当所有人都在谈论股票的时候，往往是市场即将见顶的信号。"

> "冰点期是逆向布局的最佳时机，但前提是你还有资金。保存实力，等待机会，是每个专业投资者的必修课。"

**关键语录：**

> 炒股养家曾说："强势阶段买最强龙头，敢千高位；弱势阶段低吸超跌，快进快出；淘汰阶段空仓等待新周期。"

这段话精辟地概括了不同情绪周期阶段下的操作策略，是每个短线交易者必须牢记的核心理念。

记住：情绪周期是市场情绪的轮回，理解它，跟随它，你就能在市场中立于不败之地。掌握情绪周期，就是掌握了市场的节奏密码。
