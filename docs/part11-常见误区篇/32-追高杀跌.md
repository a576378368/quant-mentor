# 第32章 误区一：追高杀跌

## 32.1 追高杀跌的危害

**炒股养家**：
> 别人贪婪时我更贪婪，别人恐慌时我更恐慌。

追高杀跌是散户最大的误区之一。追高杀跌的结果往往是高位接盘，低位割肉。

### 32.1.1 追高杀跌的3个危害

1. **高位接盘**：追高买入，高位接盘
2. **低位割肉**：恐慌割肉，低位卖出
3. **亏损扩大**：亏损不断扩大

### 32.1.2 追高杀跌的3个原因

1. **贪婪**：看到赚钱就想追
2. **恐惧**：看到亏损就想割
3. **从众**：看到别人买就跟风

## 32.2 追高的表现

### 32.2.1 表现1：龙头已涨很多 still追高

**错误做法**：

```
龙头A已经上涨50%
看到还在涨
继续追高买入
结果：高位接盘
```

**正确做法**：

```
龙头A已经上涨50%
观察龙头状态
等待确认买点
不追高
```

### 32.2.2 表现2：高位追高

**错误做法**：

```
股票高位
看到还在涨
继续追高买入
结果：高位接盘
```

**正确做法**：

```
股票高位
观察高位风险
等待确认买点
不追高
```

### 32.2.3 表现3：盲目追高

**错误做法**：

```
看到别人追高
盲目跟风
高位追高买入
结果：高位接盘
```

**正确做法**：

```
看到别人追高
冷静分析
不盲目跟风
等待确认
```

## 32.4 追高杀跌的破解方法

### 32.4.1 破解方法1：只做确认的信号

**方法**：

1. **首板确认**：只做首板确认
2. **二板确认**：只做二板确认
3. **断板回封**：只做断板回封

**例子**：

```
龙头A启动
等待确认
首板确认
买入
```

### 32.4.2 破解方法2：不贪心，不恋战

**方法**：

1. **不贪心**：不贪心买入
2. **不恋战**：不恋战卖出
3. **快进快出**：快进快出

**例子**：

```
买入确认
达到目标
立即卖出
不贪心
```

### 32.4.3 破解方法3：严格按照计划执行

**方法**：

1. **制定计划**：制定交易计划
2. **严格执行**：严格执行计划
3. **不随意改变**：不随意改变

**例子**：

```
制定计划
买入信号
严格执行
卖出信号
严格执行
```

## 32.5 实战案例
