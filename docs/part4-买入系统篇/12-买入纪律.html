第12章 买入纪律：严格执行#

12.1 买入纪律的重要性#

赵老哥：

错误的交易，第二个集合割，效果最好。忘掉他，马上进入一个新的交易状态。

买入纪律是交易的生命线。没有纪律，就无法稳定盈利。

12.1.1 为什么需要买入纪律？#

  1. 保护本金：纪律是保护本金的第一道防线

  2. 避免情绪化：纪律避免情绪化操作

  3. 提高胜率：纪律提高交易胜率

  4. 长期生存：纪律是长期生存的保障

12.1.2 买入纪律的核心#

  1. 确认信号：只做确认的信号

  2. 严格执行：严格按照计划执行

  3. 快速止损：买错立即止损

  4. 不贪不恋：不贪心，不恋战

12.2 买入纪律的4大原则#

12.2.1 原则1：只做确认的信号#

错误做法： - 猜测买点 - 听消息买入 - 盲目追高

正确做法： - 首板确认买入 - 二板确认买入 - 断板回封买入

12.2.2 原则2：严格按照计划执行#

错误做法： - 随意改变计划 - 不按计划买入 - 凭感觉操作

正确做法： - 提前制定计划 - 严格按照计划 - 不随意改变

12.2.3 原则3：买错立即止损#

错误做法： - 买错后不止损 - 等待回本 - 亏损扩大

正确做法： - 买错立即止损 - 不恋战 - 保护本金

12.2.4 原则4：不贪心不恋战#

错误做法： - 想买在最低点 - 想卖在最高点 - 深套后不止损

正确做法： - 接受不完美 - 快进快出 - 不贪心

12.3 买入纪律的具体执行#

12.3.1 买前检查清单#

执行要点： - 所有检查项都通过 ✓ 才买入 - 任何一个检查项未通过 ✗ 不买入

12.3.2 买后检查清单#

执行要点： - 每笔交易后检查 - 发现问题及时改进 - 持续优化系统

12.4 买入纪律的常见问题#

12.4.1 问题1：不按计划执行#

表现： - 随意改变计划 - 凭感觉操作 - 不按买点买入

解决方法： - 提前制定计划 - 写交易日志 - 定期复盘

12.4.2 问题2：不执行止损#

表现： - 买错后不止损 - 等待回本 - 亏损扩大

解决方法： - 设定止损位 - 到位立即执行 - 快速止损

12.4.3 问题3：情绪化操作#

表现： - 盈利时贪心 - 亏损时恐惧 - 随意改变计划

解决方法： - 保持冷静 - 严格执行纪律 - 不受情绪影响

12.4.4 问题4：不检查信号#

表现： - 听消息买入 - 盲目追高 - 猜测买点

解决方法： - 只做确认信号 - 严格检查清单 - 不凭感觉

12.5 买入纪律的修炼#

12.5.1 第一阶段：学习纪律#

内容： - 学习买入纪律 - 理解纪律重要性 - 掌握检查清单

时间：1-2周

12.5.2 第二阶段：模拟练习#

内容： - 模拟交易练习 - 应用检查清单 - 练习严格执行

时间：1-2个月

12.5.3 第三阶段：小仓实盘#

内容： - 小仓实盘操作 - 严格执行纪律 - 记录交易日志

时间：2-3个月

12.5.4 第四阶段：稳定盈利#

内容： - 稳定盈利 - 习惯纪律 - 自动执行

时间：3-6个月

12.6 实战案例#

12.6.1 案例1：严格执行纪律#

时间：2026年3月 题材：人工智能 股票：科技龙头A（00XX）

操作过程：

  1. 买前检查： - 市场环境：强势 ✓ - 题材确认：新题材 ✓ - 龙头确认：龙头股 ✓ - 买点确认：首板 ✓ - 仓位确认：70% ✓ - 止损确认：明确 ✓

  2. 买后执行： - 策略执行：按计划 ✓ - 纪律遵守：严格执行 ✓ - 情绪控制：冷静 ✓ - 仓位管理：70% ✓ - 止损执行：明确 ✓

  3. 结果： - 涨幅25% - 盈利目标达成 - 高潮分歧止盈

12.6.2 案例2：违反纪律的后果#

时间：2026年4月 题材：新能源 股票：新能源B（00XX）

操作过程：

  1. 买前检查： - 市场环境：弱势 ✓ - 题材确认：旧题材 ✗ - 龙头确认：非龙头 ✗ - 买点确认：高位追高 ✗ - 仓位确认：80% ✗ - 止损确认：未明确 ✗

  2. 买后结果： - 股票连续下跌 - 亏损20% - 未及时止损 - 亏损扩大

教训： - 违反纪律 - 亏损扩大 - 及时止损

12.7 小结#

买入纪律：

  1. 4大原则：只做确认信号、严格执行计划、买错立即止损、不贪心不恋战

  2. 2个清单：买前检查清单、买后检查清单

  3. 4个问题：不按计划、不执行止损、情绪化、不检查信号

  4. 4个阶段：学习纪律、模拟练习、小仓实盘、稳定盈利

记住：纪律是交易的生命线。没有纪律，就无法稳定盈利。严格执行纪律，是长期生存的保障。
