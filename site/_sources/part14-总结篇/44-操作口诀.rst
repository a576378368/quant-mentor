第44章 操作口诀
=========


44.1 操作口诀总览
-----------


**炒股养家**：
   得散户心者得天下。人气所向，牛股所在。

操作口诀是游资心法的精华。记住这些口诀，就能在市场中游刃有余。

44.2 核心口诀
---------


44.2.1 龙头口诀
-----------


::

    强者恒强，弱者恒弱
    龙头断板，立即卖出
    板块退潮，全部清仓
    赢面不足，空仓观望
    风控触发，立即止损


**解读**：

1. **强者恒强**：龙头股强者恒强
2. **弱者恒弱**：跟风股弱者恒弱
3. **龙头断板**：龙头断板立即卖出
4. **板块退潮**：板块退潮全部清仓
5. **赢面不足**：赢面不足空仓观望
6. **风控触发**：风控触发立即止损

44.2.2 买入口诀
-----------


::

    只做确认，不做猜测
    首板突破，二板确认
    断板回封，龙头确认
    小仓试错，大仓持有
    买错立即，止损离场


**解读**：

1. **只做确认**：只做确认的信号
2. **不做猜测**：不做猜测
3. **首板突破**：首板突破买入
4. **二板确认**：二板确认买入
5. **断板回封**：断板回封买入
6. **小仓试错**：小仓试错
7. **大仓持有**：大仓持有
8. **买错立即**：买错立即止损

44.2.3 卖出口诀
-----------


````>
龙头断板，立即卖出
板块退潮，全部卖出
达到目标，分批卖出
风控触发，立即止损
不贪不恋，快进快出
::

    
    **解读**：
    
    1. **龙头断板**：龙头断板立即卖出
    2. **板块退潮**：板块退潮全部卖出
    3. **达到目标**：达到目标分批卖出
    4. **风控触发**：风控触发立即止损
    5. **不贪不恋**：不贪不恋
    6. **快进快出**：快进快出
    
    ### 44.2.4 仓位口诀
    

赢面低，仓位小
赢面高，仓位大
盈亏比高，仓位大
盈亏比低，仓位小
60%以下，空仓观望
::

    
    **解读**：
    
    1. **赢面低**：赢面低仓位小
    2. **赢面高**：赢面高仓位大
    3. **盈亏比高**：盈亏比高仓位大
    4. **盈亏比低**：盈亏比低仓位小
    5. **60%以下**：60%以下空仓观望
    
    ## 44.3 风控口诀
    

单日3%，立即停止
连续3次，暂停1天
最大10%，重置仓位
仓位95%，留出余量
行业30%，分散风险
::

    
    **解读**：
    
    1. **单日3%**：单日亏损3%立即停止
    2. **连续3次**：连续亏损3次暂停1天
    3. **最大10%**：最大回撤10%重置仓位
    4. **仓位95%**：仓位上限95%
    5. **行业30%**：行业集中度30%
    
    ## 44.4 心态口诀
    

信念第一，心态第二
不贪不恋，不慌不忙
人多不去，绝望布局
持续学习，更高境界
::

    
    **解读**：
    
    1. **信念第一**：信念第一
    2. **心态第二**：心态第二
    3. **不贪不恋**：不贪不恋
    4. **不慌不忙**：不慌不忙
    5. **人多不去**：人多的地方不去
    6. **绝望布局**：绝望时布局
    7. **持续学习**：持续学习
    8. **更高境界**：追求更高境界
    
    ## 44.5 小结
    
    操作口诀：
    
    1. **5大口诀**：龙头口诀、买入口诀、卖出口诀、仓位口诀、风控口诀
    2. **4个要点**：只做确认、龙头至 上、不贪不恋、风控第一
    
    记住：操作口诀是游资心法的精华。记住这些口诀，就能在市场中游刃有余。
