    第43章 核心思想

    43.1 核心思想总览

    炒股养家：

    市场是财富再分配场所，本质是人性博弈。

    游资心法的核心思想是建立一套可持续的盈利系统。这套系统不是赌博，而是一门职业，需要信念、纪律、和系统。

    43.1.1 核心思想的5大支柱

    1. 市场本质：市场是财富再分配场所，本质是人性博弈
    2. 情绪周期：掌握情绪周期变化，不同阶段采用不同策略
    3. 龙头战法：只做龙头，只做主升，只做惯性
    4. 仓位管理：根据赢面动态调整仓位
    5. 风控原则：保护本金第一，严格风控

    43.2 市场本质

    43.2.1 财富再分配

    炒股养家：

    市场是财富再分配场所，本质是人性博弈。

    解读：

    1. 短期：资金从一部分人手中转移到另一部分人手中
    2. 中期：资金从情绪化交易者转移到理性交易者手中
    3. 长期：价值回归，但A股不适合长期持有

    43.2.2 人性博弈

    人性的3个表现：

    1. 贪婪：看到赚钱就想追高，导致高位接盘
    2. 恐惧：看到亏损就恐慌割肉，导致亏损扩大
    3. 从众：看到别人赚钱就盲目跟风，成为接盘侠

    43.3 情绪周期

    43.3.1 8阶段情绪周期

    酝酿 → 起步 → 加速 → 高潮 → 发散 → 退潮 → 淘汰 → 新周期

    每个阶段的操作策略：

    阶段 操作策略
    酝酿 空仓等待，或小仓试错
    起步 小仓试错，确认后加仓
    加速 主升段持有，回调加仓
    高潮 分批止盈，不贪心
    发散 切换新题材，不恋战
    退潮 清仓观望，不抄底
    淘汰 空仓等待，不操作
    新周期 小仓试错，确认后跟进

    43.3.2 冰点高潮

    冰点： - 涨停数 < 10 - 跌停数 > 50 - 连板数 < 3 - 涨跌比 < 0.5

    高潮： - 涨停数 > 100 - 涨跌比 > 2 - 连板数增加 - 炸板率 < 20%

    43.4 龙头战法

    43.4.1 龙头标准

    1. 位置最低：第一个涨停
    2. 资金介入：成交量 > 5日均量 2倍
    3. 板块带动：带动同板块股票 >= 2只
    4. 市场关注：龙虎榜有游资席位
    5. 爆量验证：必须经过市场检验

    43.4.2 龙头战法要点

    1. 发现龙头：第一时间发现市场精神领袖
    2. 确认龙头：第一时间上了她
    3. 持有龙头：一直持有，直到发现市场新的精神领袖果断切换
    4. 爆量验证：没爆量的都不能说是龙头

    43.5 仓位管理

    43.5.1 仓位计算公式

    基础仓位 = 100% × 赢面 × 上涨空间 / (上涨空间 + 下跌空间)

    例子：

    胜率80%，上涨空间30%，下跌空间5%
基础仓位 = 100 × 0.8 × 30 / (30+5) ≈ 69%

    43.5.2 仓位分级

    赢面 胜率 操作 仓位
    <60% <70% 观望 0%
    60-70% 70-80% 小仓 20-30%
    70-80% 80-90% 中仓 40-60%
    80-90% 90-95% 大仓 70-85%
    ≥90% ≥95% 满仓 90-95%

    43.6 风控原则

    43.6.1 风控指标

    • 单日最大亏损：3%
    • 连续亏损：3次（暂停交易1天）
    • 最大回撤：10%（重置仓位）
    • 仓位上限：95%
    • 行业集中度：30%

    43.6.2 风控措施

    单日亏损 > 3%： - 措施：当日剩余时间停止交易 - 恢复：下一个交易日

    连续亏损 >= 3次： - 措施：暂停交易1天 - 恢复：24小时后

    最大回撤 >= 10%： - 措施：仓位重置到50% - 恢复：手动确认

    43.7 小结

    核心思想：

    1. 5大支柱：市场本质、情绪周期、龙头战法、仓位管理、风控原则
    2. 市场本质：财富再分配，人性博弈
    3. 情绪周期：8阶段，不同阶段不同策略
    4. 龙头战法：只做龙头，只做主升，只做惯性
    5. 仓位管理：根据赢面动态调整
    6. 风控原则：保护本金第一

    记住：核心思想是游资成功的基石。掌握市场本质、情绪周期、龙头战法、仓位管理、风控原则，才能在市场中立于不败之地。
