    第21章 风险管理：生存第一的完整体系¶

    21.1 风险管理的交易哲学¶

    21.1.1 风险管理的本质¶

    炒股养家曾说：「最重要的是信念二字。关于心态，股票做好了，能实现稳定盈利，心态也就不是问题了。」这句话深刻揭示了风险管理的本质——风险管理不仅是技术问题，更是心态和信念的问题。

    风险管理是交易系统的最高层。它不是某一项具体的技巧或指标，而是贯穿整个交易系统、涵盖所有交易环节的完整体系。如果说风控指标是安全带的扣环，风控措施是安全带的拉伸机制，那么风险管理就是整辆汽车的安全系统设计——它决定了为什么要有安全带、如何设计安全带、以及在什么情况下安全带应该发挥作用。

    很多交易者失败的原因，不是他们不懂风险管理，而是他们不能真正理解风险管理的核心——生存第一。这个看似简单的道理，却让无数聪明人折戟沉沙。当账户出现盈利时，贪婪会让人忘记风险；当账户出现亏损时，侥幸会让人忽视风险。风险管理要做的，就是在这两种情况下都保持清醒，确保生存永远是第一优先级。

    徐翔对此有深刻理解：「在这个市场上，永远不要试图赚取最后一块铜板。」这句话不仅是止损的理念，更是风险管理的核心理念——不要为了追求最高收益而承担过度的风险。风险管理追求的是长期稳定盈利，而不是某一次的暴利。

    21.1.2 风险管理的三个层次¶

    风险管理分为三个层次，每个层次有不同的重点和方法。

    第一层：战术层面的风险管理

    战术层面的风险管理针对单笔交易，指的是在每一笔交易中进行风险控制。

    核心要点： 

    1. 单笔交易最大亏损不超过账户的2%
2. 每次买入前必须设定止损位
3. 止损位一旦设定，必须严格执行
4. 单笔交易的仓位必须计算精确

    战术层面风险管理的目的是：确保每一笔交易的风险都是可控的，不会因为单笔交易的亏损而影响账户的整体安全。

    第二层：策略层面的风险管理

    策略层面的风险管理针对整个交易系统，指的是在一定时间内（通常是一个月或一个季度）评估交易系统的表现。

    核心要点： 

    1. 交易系统的胜率必须大于50%
2. 交易系统的盈亏比必须大于2:1
3. 交易系统的最大连续亏损次数必须有限制
4. 交易系统的期望收益必须为正

    策略层面风险管理的目的是：确保交易系统本身是有效的，能够在长期内产生正期望收益。

    第三层：账户层面的风险管理

    账户层面的风险管理针对整个账户，指的是管理账户的整体风险暴露。

    核心要点： 

    1. 单日最大亏损不超过账户总值的3%
2. 最大回撤不超过账户峰值的10%
3. 连续亏损次数超过3次必须暂停交易
4. 仓位永远不要超过95%

    账户层面风险管理的目的是：确保账户在任何情况下都不会遭受毁灭性打击，能够长期生存。

    21.1.3 风险管理的三大目标¶

    目标一：生存第一

    生存第一是风险管理的首要目标。这个目标看似简单，却需要交易者时刻保持警惕。

    生存第一的含义： 

    1. 任何时候都不能让账户遭受毁灭性打击
2. 任何时候都要留有继续交易的资金
3. 任何时候都不能因为一笔交易而影响整体
4. 长期生存比短期暴利更重要

    生存第一的实现方法： 

    1. 设置严格的风控指标（单日亏损3%、最大回撤10%）
2. 执行坚定的风控措施（触发即执行）
3. 保持合理的仓位（永远不超过95%）
4. 分散投资（行业分散、个股分散）

    目标二：稳定盈利

    在确保生存的前提下，风险管理的第二个目标是实现稳定盈利。

    稳定盈利的含义： 

    1. 不是追求每次都盈利，而是追求长期整体盈利
2. 不是追求短期暴利，而是追求长期稳定收益
3. 不是追求抓住所有机会，而是追求有效把握确定性机会

    稳定盈利的实现方法： 

    1. 建立高胜率的交易系统（胜率 > 50%）
2. 确保盈亏比有利（盈亏比 > 2:1）
3. 控制交易频率（不做无谓的交易）
4. 持续优化交易系统（不断改进）

    目标三：资金曲线平滑

    风险管理的第三个目标是保持资金曲线的平滑，避免大起大落。

    资金曲线平滑的含义： 

    1. 资金曲线呈现稳定上升趋势
2. 没有剧烈的大幅回撤
3. 没有长时间的横盘或下跌
4. 夏普比率较高

    资金曲线平滑的实现方法： 

    1. 严格执行风控指标（不追求一把暴富）
2. 均衡配置仓位（不重仓单一个股）
3. 分散投资（行业分散、策略分散）
4. 动态调整（根据市场环境调整仓位）

    21.2 单笔交易的风险管理¶

    21.2.1 单笔交易风险的计算¶

    单笔交易风险管理的第一步是精确计算单笔交易的风险。

    风险计算公式： 

    单笔交易风险金额 = 账户总资产 × 单笔最大亏损比例

单笔交易最大亏损比例建议：2%（保守）或 3%（标准）

例如：
账户总资产：100万
单笔最大亏损比例：2%
单笔交易风险金额：2万

    止损位计算： 

    止损位 = 买入价 × 止损比例

止损比例建议：5%-10%（根据股票波动性调整）

例如：
买入价：10元
止损比例：8%
止损位：9.2元
单股亏损：0.8元

    仓位计算： 

    最大买入股数 = 单笔交易风险金额 ÷ 单股亏损金额

例如：
单笔交易风险金额：2万
单股亏损：0.8元
最大买入股数：2万 ÷ 0.8元 = 2.5万股

买入市值：2.5万股 × 10元 = 25万
仓位比例：25万 ÷ 100万 = 25%

    21.2.2 单笔交易风险控制的要点¶

    要点一：买入前必须计算风险

    在任何买入决策之前，必须先计算好该笔交易的风险：

    买入前检查清单：
□ 账户总资产：____万
□ 单笔最大亏损：____万（总资产的2%）
□ 买入价格：____元
□ 止损价格：____元
□ 单股亏损：____元
□ 最大买入股数：____股
□ 买入市值：____万
□ 仓位比例：____%

检查结论：
- 如果仓位比例 > 30%，降低买入数量
- 如果止损空间 > 10%，重新评估是否买入
- 如果无法计算风险金额，禁止买入

    要点二：设置止损位必须合理

    止损位的设置是单笔交易风险控制的核心：

    止损位设置原则：

1. 必须有明显支撑位：
   - 技术支撑位（均线、平台、前低）
   - 整数关口
   - 成交量密集区

2. 必须考虑股票波动性：
   - 波动大的股票，止损空间适当放大
   - 波动小的股票，止损空间可以缩小

3. 必须符合资金管理要求：
   - 单笔亏损不能超过账户的2%
   - 止损空间 × 股数 = 风险金额

止损位设置示例：
股票A：
- 买入价：10元
- 波动性：高（日常波动5-8%）
- 技术支撑：9.5元（前低）
- 止损位：9.5元
- 止损空间：5%
- 单股亏损：0.5元
- 最大买入：2万 ÷ 0.5 = 4万股
- 仓位：4万股 × 10元 = 40%（略高，降低到3万股，仓位30%）

股票B：
- 买入价：20元
- 波动性：低（日常波动2-3%）
- 技术支撑：19.2元（均线支撑）
- 止损位：19.2元
- 止损空间：4%
- 单股亏损：0.8元
- 最大买入：2万 ÷ 0.8 = 2.5万股
- 仓位：2.5万股 × 20元 = 50%（超过30%，降低到1万股，仓位10%）

    要点三：止损执行必须坚决

    止损是单笔交易风险控制的最后一道防线：

    止损执行原则：

1. 到达止损位，无条件执行
   - 不考虑是否可能反弹
   - 不考虑是否已经亏了很多
   - 不考虑别人的看法

2. 止损执行要快
   - 触发止损后立即执行
   - 不要犹豫、不要等待
   - 执行时不要看行情

3. 止损后不要立即反手
   - 止损说明当前方向错误
   - 需要等待趋势明朗
   - 至少观察1-2个交易日

止损执行检查清单：
□ 持仓是否到达止损位？
   - 是 → 执行止损
   - 否 → 继续持有

□ 执行止损时是否犹豫？
   - 是 → 加强执行力训练
   - 否 → 继续

□ 止损后是否立即想反手？
   - 是 → 禁止，强制等待
   - 否 → 完成止损

    21.2.3 单笔交易风险管理案例¶

    案例一：追高买入的风险控制

    背景： - 账户：100万 - 市场：强势上涨 - 标的：科技龙头股A

    买入决策过程： 

    股价当天涨停板打开后回落，买入价：45元
此时想买，但需要计算风险

风险计算：
1. 止损位设置：42元（回调3元，6.7%空间）
2. 单股亏损：3元
3. 单笔最大亏损：2万（账户的2%）
4. 最大买入股数：2万 ÷ 3元 = 6666股
5. 买入市值：6666股 × 45元 = 30万
6. 仓位：30%

分析结论：
- 仓位30%在合理范围
- 止损空间6.7%略大，但在可接受范围
- 可以买入，但仓位控制在30%

执行结果：
- 买入6666股，市值30万
- 设置止损价42元
- 亏损控制在下限内

    案例二：弱势股票的止损执行

    背景： - 账户：95万（前期亏损5万） - 市场：弱势震荡 - 持仓：消费股B，买入价10元，现价9.2元

    持仓状态： 

    买入价：10元
当前价：9.2元
浮动亏损：0.8元（8%）
持仓数量：3万股
浮亏金额：2.4万（2.5%）
账户总资产：95万
单笔最大亏损限额：1.9万（95万的2%）

已接近单笔亏损上限，需要考虑止损

    止损决策： 

    止损分析：
1. 当前亏损2.4万，已超过单笔最大亏损1.9万
2. 到达自己设定的止损位（9.2元 > 9元支撑位，尚未破位）
3. 但已接近单笔亏损上限，需要决定是否止损

决策：
- 不等到达止损位，提前止损
- 卖出价：9.1元（略高于支撑位）
- 亏损：0.9元 × 3万股 = 2.7万
- 占账户比例：2.7%

执行止损的理由：
1. 已超过单笔最大亏损限制
2. 市场弱势，可能继续下跌
3. 需要保留资金等待更好的机会

    执行结果： 

    执行时间：14:00
卖出价格：9.1元
卖出数量：3万股
实际亏损：2.7万（2.8%）
账户剩余：92.3万

反思：
1. 买入时风险计算不够精确
2. 止损位设置不够合理
3. 需要加强弱势市场的风险控制

    21.3 策略层面的风险管理¶

    21.3.1 交易系统的有效性验证¶

    策略层面的风险管理首先需要验证交易系统的有效性。

    胜率验证：

    胜率是交易系统有效性的基础指标： 

    胜率定义：盈利交易次数 ÷ 总交易次数

胜率要求：> 50%（最好 > 60%）

胜率计算方法：
统计周期：至少100笔交易
统计公式：胜率 = 盈利交易次数 ÷ 总交易次数 × 100%

示例：
100笔交易中，盈利交易60笔
胜率 = 60 ÷ 100 × 100% = 60%

胜率评估标准：
- 胜率 < 40%：系统无效，需要改进
- 胜率 40%-50%：系统勉强有效，但需要优化
- 胜率 50%-60%：系统有效，可继续使用
- 胜率 > 60%：系统优秀，盈利能力较强

    盈亏比验证：

    盈亏比是交易系统有效性的关键指标： 

    盈亏比定义：平均盈利金额 ÷ 平均亏损金额

盈亏比要求：> 2:1（最好 > 3:1）

盈亏比计算方法：
平均盈利 = 盈利总额 ÷ 盈利交易次数
平均亏损 = 亏损总额 ÷ 亏损交易次数
盈亏比 = 平均盈利 ÷ 平均亏损

示例：
盈利交易40笔，总盈利80万，平均盈利2万
亏损交易60笔，总亏损60万，平均亏损1万
盈亏比 = 2万 ÷ 1万 = 2:1

盈亏比评估标准：
- 盈亏比 < 1.5:1：系统风险较高，需要改进
- 盈亏比 1.5:1-2:1：系统可接受，但需要优化
- 盈亏比 2:1-3:1：系统有效，盈利能力较强
- 盈亏比 > 3:1：系统优秀，盈利能力很强

    期望收益验证：

    期望收益是交易系统有效性的综合指标： 

    期望收益定义：每笔交易的平均预期收益

期望收益计算公式：
E = 胜率 × 平均盈利 - (1 - 胜率) × 平均亏损

或者：
E = 胜率 × 盈亏比 - (1 - 胜率)

期望收益评估标准：
- E < 0：系统无效，亏损
- E = 0-0.5：系统勉强有效
- E = 0.5-1.0：系统有效
- E > 1.0：系统优秀

示例：
胜率60%，盈亏比2:1
E = 0.6 × 2 - 0.4 × 1 = 1.2 - 0.4 = 0.8
期望收益为正，系统有效

    21.3.2 交易系统的风险评估¶

    除了验证交易系统的有效性，还需要评估交易系统的风险特征。

    最大连续亏损评估：

    最大连续亏损是交易系统风险的重要指标： 

    最大连续亏损定义：交易系统在最不利情况下可能出现的连续亏损

计算方法：
历史数据统计：统计过去N笔交易中的最大连续亏损次数和金额
理论估算：根据胜率和盈亏比估算最大连续亏损

理论估算公式：
假设胜率60%，连续亏损5次的概率 = (1-0.6)^5 = 0.4^5 = 1.024%
假设每次亏损1万，连续亏损5次 = 5万

评估标准：
- 最大连续亏损 < 5次：风险较低
- 最大连续亏损 5-8次：风险中等
- 最大连续亏损 > 8次：风险较高

    最大回撤评估：

    最大回撤是交易系统风险的另一个重要指标： 

    最大回撤定义：交易系统从历史最高点到最低点的最大跌幅

计算方法：
最大回撤 = (历史最高值 - 历史最低值) ÷ 历史最高值 × 100%

评估标准：
- 最大回撤 < 10%：风险较低
- 最大回撤 10%-20%：风险中等
- 最大回撤 > 20%：风险较高

    夏普比率评估：

    夏普比率是衡量交易系统风险收益比的标准指标： 

    夏普比率定义：(平均收益率 - 无风险收益率) ÷ 收益率标准差

计算方法：
需要至少1年以上的历史数据
建议使用月度收益率计算

评估标准：
- 夏普比率 < 0.5：风险收益比较低
- 夏普比率 0.5-1.0：风险收益比中等
- 夏普比率 1.0-2.0：风险收益比较高
- 夏普比率 > 2.0：风险收益比很高

    21.3.3 交易系统的动态调整¶

    交易系统需要根据市场环境进行动态调整：

    市场环境分类： 

    环境分类：
1. 强势市场（牛市氛围）：
   - 大盘趋势向上
   - 赚钱效应强
   - 涨停家数多
   - 操作策略：积极进攻，提高仓位

   - 亏钱效应强
   - 跌停家数多
   - 操作策略：防守为主，降低仓位

3. 震荡市场：
   - 大盘横盘整理
   - 热点快速轮动
   - 操作策略：均衡配置，高抛低吸

    不同环境下的策略调整： 

    强势市场策略调整：
- 仓位：60% - 80%（提高）
- 止损：适当放宽（8%-10%）
- 止盈：适当提高（15%-20%）
- 交易频率：可以增加

弱势市场策略调整：
- 仓位：30% - 50%（降低）
- 止损：更加严格（5%-6%）
- 止盈：适当降低（8%-10%）
- 交易频率：应该减少

震荡市场策略调整：
- 仓位：40% - 60%（中等）
- 止损：正常（6%-8%）
- 止盈：正常（10%-15%）
- 交易频率：中等

    单日亏损3%是账户层面的第一条红线： 

    触发条件：当日亏损 ≥ 账户总资产的3%

例如：
账户总资产：100万
单日最大亏损：3万

执行动作：
1. 当日立即停止所有交易
2. 清空所有待执行订单
3. 持仓保持不变
4. 记录当日交易情况
5. 分析亏损原因
6. 次日恢复交易（需满足条件）

    最大回撤控制（10%）：

    最大回撤10%是账户层面的第二条红线： 

    触发条件：从历史最高资产回撤 ≥ 10%

例如：
历史最高资产：105万
当前资产：94.5万
回撤：(105 - 94.5) / 105 = 10%

执行动作：
1. 当日收盘前将仓位降至50%
2. 至少3个交易日内不增加仓位
3. 分析回撤原因
4. 重新评估市场环境
5. 逐步恢复仓位

    连续亏损控制（3次）：

    连续亏损3次是账户层面的第三条红线： 

    触发条件：连续3笔交易亏损

执行动作：
1. 当日停止交易
2. 深度复盘分析
3. 制定改进措施
4. 次日暂停交易1天
5. 后天恢复交易（从50%仓位开始）

    21.4.3 账户层面的应急预案¶

    账户层面需要建立完善的应急预案，以应对各种极端情况：

    情景一：市场连续大跌

    触发条件：大盘连续下跌超过10% 

    应急预案：
1. 开盘前分析市场情况
2. 检查持仓股的风险
3. 如果持仓股有继续下跌风险，果断减仓
4. 仓位控制在30%以内
5. 不要急于抄底，等待市场企稳
6. 企稳后逐步加仓

    情景二：持仓股连续跌停

    触发条件：持仓股连续2天跌停 

    应急预案：
1. 第二天开盘前挂单卖出
2. 如果开盘仍是跌停，耐心等待盘中打开
3. 打开时果断卖出，不要考虑价格
4. 卖出后至少观察3个交易日
5. 不要立即补仓或买入其他股票
6. 分析该股的基本面是否发生变化

    情景三：账户大幅回撤

    触发条件：账户回撤超过15% 

    应急预案：
1. 当日将仓位降至20%以下
2. 分析回撤原因
3. 暂停交易至少1周
4. 在休息期间进行系统学习
5. 休息后从小仓位开始恢复
6. 逐步增加仓位，不要急于把损失扳回来

    情景四：系统性风险爆发

    触发条件：出现重大利空（政策、国际市场等） 

    应急预案：
1. 第一时间评估影响程度
2. 如果影响严重，立即减仓
3. 不要抱有侥幸心理
4. 保护本金第一
5. 等待风险释放后再考虑入场
6. 不要试图接飞刀

    21.5 风险管理工具详解¶

    21.5.1 止损单的使用¶

    止损单是风险管理最基本的工具：

    止损单的种类：

    1. 固定止损单 

      定义：价格达到预设价位时自动卖出
设置方法：买入时设定止损价格
优点：简单明了，执行确定
缺点：可能因为正常波动被触发

使用场景：
- 适合波动较小的股票
- 适合中长期持仓
- 适合稳健型交易者

    2. 移动止损单 

      定义：随着股价上涨，逐步提高止损价格
设置方法：设定一个固定百分比或金额
优点：锁定已有利润
缺点：可能因为正常回调被触发

使用场景：
- 适合强势股
- 适合趋势明确的股票
- 适合想要持有趋势的交易者

示例：
买入价：10元
初始止损：9元（下跌10%）
股价涨到12元时，止损提高到10.8元（下跌10%）
股价涨到15元时，止损提高到13.5元（下跌10%）

    3. 时间止损单 

      定义：持仓超过一定时间后强制卖出
设置方法：设定持仓天数
优点：避免长期持有不确定性
缺点：可能错过大机会

使用场景：
- 适合短线交易
- 适合题材炒作
- 适合明确时间窗口的机会

    止损单设置技巧：

    技巧一：设置在明显支撑位
- 均线支撑
- 前期低点
- 平台支撑

技巧二：留出正常波动空间
- 不要设置太紧
- 给股票正常的波动空间
- 但也不能太松，失去止损意义

技巧三：根据波动性调整
- 波动大的股票，止损空间适当放大
- 波动小的股票，止损空间可以缩小

技巧四：分批设置止损
- 第一次止损：稍微宽松
- 第二次止损：相对严格
- 目的：保留头寸的同时控制风险

    21.6 泽熙的风险管理体系¶

    21.6.1 徐翔的风险管理哲学¶

    徐翔是A股市场最成功的交易者之一，他的风险管理哲学值得深入研究：

    核心理念一：生存第一

    徐翔始终将生存放在第一位：「在这个市场上，活得久才是王道。」

    他的具体做法： 

    1. 任何时候都不重仓单一股票
2. 任何时候都不满仓操作
3. 任何时候都留有现金
4. 任何时候都不冒毁灭性风险

    核心理念二：止损是本能

    徐翔认为止损是交易者的本能：「止损不是靠意志力，而是靠系统。当触发止损条件时，系统会自动执行，不需要交易者做决定。」

    他的具体做法： 

    1. 买入时就设定好止损位
2. 止损位一旦触发，无条件执行
3. 从不止损后立即反手
4. 把止损当成交易成本

    核心理念三：控制回撤是关键

    徐翔最重视的是控制回撤：「一次大的回撤可能让你多年无法翻身。控制回撤比追求高收益更重要。」

    他的具体做法： 

    1. 设置多重回撤控制线
2. 回撤达到8%发出预警
3. 回撤达到10%减仓50%
4. 回撤达到15%必须空仓

    核心理念四：只在有把握时出手

    徐翔强调只在有把握时出手：「宁可错过，不要做错。没有把握的交易不如不做。」

    他的具体做法： 

    1. 没有明确的信号不开仓
2. 没有确认的趋势不重仓
3. 没有足够的安全边际不买入
4. 没有明确的机会不交易

    21.6.2 泽熙的仓位管理原则¶

    泽熙投资的仓位管理是业内典范：

    原则一：永远不满仓

    泽熙从不满仓操作，始终保留一定的现金： 

    仓位配置原则：

    应急流程：

2. 如果持仓股有跌停趋势，开盘立即挂单卖出
3. 如果持仓股已跌停，盘中打开时果断卖出
4. 如果市场极度恐慌，保留部分仓位等待反弹

阶段二（暴跌后3个交易日）：
1. 不轻易抄底，等待市场企稳
2. 观察持仓股是否有流动性问题
3. 如果持仓股持续下跌，考虑进一步减仓
4. 仓位控制在30%以内

1. 逐步建仓优质股票
2. 把握错杀机会
3. 等待市场恢复正常

    情景二：持仓股突发利空应急预案

    应急流程：

1. 第一时间评估影响程度
2. 如果影响严重，立即挂单卖出

1. 至少观察5个交易日
2. 分析该股的基本面是否发生根本性变化
3. 不急于补仓或买入其他股票
4. 等待市场情绪稳定

阶段三（重新评估）：
1. 根据基本面决定是否重新买入
2. 如果重新买入，仓位控制在15%以内
3. 设置更严格的止损位

    情景三：账户触发风控应急预案

2. 评估市场环境
3. 如果市场弱势，观望为主
4. 如果市场强势，从小仓位开始

阶段三（后续交易日）：
1. 仓位控制在50%以内
2. 只做高确定性机会
3. 连续盈利3次后再逐步加仓

    情景四：系统性风险应急预案

    触发条件：出现重大政策利空、国际市场暴跌、全球金融危机等系统性风险

    应急流程：

    21.8 风险管理的常见误区¶

    21.8.1 误区一：忽视风险管理¶

    错误表现：

    很多新手交易者忽视风险管理，认为只要技术好就能赚钱：

    错误观点1："我技术好，不需要风控"
- 实际情况：再好的技术也会出错
- 后果：一次大的失误可能导致毁灭性亏损

错误观点2："风控会影响盈利"
- 实际情况：风控是保护盈利的手段
- 后果：没有风控可能一次亏损就吃掉所有利润

错误观点3："我资金小，不需要风控"
- 实际情况：资金越小越要控制风险
- 后果：小资金更容易被大的亏损摧毁

    正确做法：

    风险管理要点：
1. 任何资金规模都需要风控
2. 风控是保护本金和盈利的工具
3. 风险管理比交易技术更重要
4. 先学会风控，再谈赚钱

    21.8.2 误区二：贪心不顾风险¶

    错误表现：

    很多交易者在盈利时变得贪婪，忽视风险：

    错误做法1：盈利时提高仓位
- 原本30%仓位，盈利后加到60%
- 风险扩大一倍
- 可能把利润亏回去

错误做法2：盈利时不设止损
- 觉得已经盈利了，止损无所谓
- 结果盈利变亏损
- 得不偿失

错误做法3：追求高收益忽视风险
- 重仓单一股票
- 追涨强势股
- 结果高位被套

    正确做法：

    风险管理要点：
1. 盈利时更要控制风险
2. 盈利时仓位可以适当提高，但不能过高
3. 任何时候都要设止损
4. 追求稳定盈利而不是暴利

    21.8.3 误区三：恐惧导致过度保守¶

    错误表现：

    有些交易者因为亏损而变得过于保守：

    错误做法1：仓位过低
- 账户50万，只买5万（10%）
- 机会来了也抓不住
- 收益太低

错误做法2：不敢买入
- 看到机会也不敢买
- 总是观望等待
- 错过大好机会

错误做法3：止损设置过窄
- 止损设置1-2%
- 正常波动就被止损
- 反而亏钱

    正确做法：

    风险管理要点：
1. 仓位要合理（30-60%为宜）
2. 机会来时要敢于把握
3. 止损设置要合理（5-10%）
4. 在风险和机会之间找到平衡

    21.8.4 误区四：不执行风控¶

    错误表现：

    很多交易者设了风控但不执行：

    错误做法1：设了止损不执行
- 到达止损位想着再等等
- 结果亏损继续扩大
- 小亏变大亏

错误做法2：触发风控不执行
- 触发连续亏损风控还继续交易
- 触发最大回撤风控也不减仓
- 结果亏损一发不可收拾

错误做法3：不记录风控执行情况
- 不记录就无法追溯
- 无法改进
- 同样的错误反复发生

    正确做法：

    风险管理要点：
1. 设了风控就必须执行
2. 触发风控后无条件执行
3. 把风控当成铁的纪律
4. 记录每次风控执行情况，便于复盘
