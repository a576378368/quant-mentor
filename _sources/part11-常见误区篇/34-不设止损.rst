第34章 误区三：不设止损
=============


34.1 不设止损的危害
------------


**炒股养家**：
   出局就是出局，不管止损止盈。

不设止损是散户最大的风险之一。不设止损的结果往往是深套后亏损扩大。

34.1.1 不设止损的3个危害
----------------


1. **深套后不止损**：深套后不止损，亏损扩大
2. **期望回本**：期望回本，不及时止损
3. **亏损扩大**：亏损不断扩大

34.1.2 不设止损的3个原因
----------------


1. **侥幸心理**：认为能回本
2. **恐惧心理**：害怕确认亏损
3. **贪心心理**：希望继续上涨

34.2 不设止损的表现
------------


34.2.1 表现1：深套后 not卖出
--------------------


**错误做法**：

::

    股票深套30%
    不卖出
    期望回本
    结果：亏损扩大


**正确做法**：

::

    股票深套30%
    立即止损
    保护本金


34.2.2 表现2：期望回本
---------------


**错误做法**：

::

    股票深套
    期望回本
    不卖出
    结果：亏损扩大


**正确做法**：

::

    股票深套
    立即止损
    不期望回本


34.2.3 表现3：深套不止损
----------------


**错误做法**：

::

    股票深套
    不止损
    亏损扩大
    结果：更大亏损


**正确做法**：

::

    股票深套
    立即止损
    不恋战

34.4 实战案例
---------


34.4.1 案例1：不设止损失败
-----------------


**时间**：2026年3月  
**股票**：科技A（00XX）

**不设止损过程**：

1. **买入**：
   - 科技A买入
   - 买入价10元
   - 不设止损

2. **深套**：
   - 科技A下跌30%
   - 不卖出
   - 期望回本

3. **结果**：
   - 盈利：-30%
   - 不设止损
   - 亏损扩大

34.4.2 案例2：设止损成功
----------------


**时间**：2026年4月  
**股票**：新能源A（00XX）

**设止损过程**：

1. **买入**：
   - 新能源A买入
   - 买入价10元
   - 设止损价9元

2. **止损**：
   - 新能源A下跌到9元
   - 触发止损
   - 立即卖出

3. **结果**：
   - 亏损：10%
   - 设止损
   - 保护本金
